## 🧑‍💻 Chi sono

Ciao! Mi chiamo **Lorenzo**, e sono il creatore di **PrimeBuild**, un progetto nato per offrire:
- 🧩 **Assemblaggi PC** ad alte prestazioni
- 🧠 **Consulenza e tweaking** per eSport
